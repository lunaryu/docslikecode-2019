文档代码化，想说爱你不容易¶

三天的培训一眨眼，两天就要过去了。第一次接触代码，很兴奋。

技能get¶

了解了文档代码化的基本概念及原理，收获很大。学习了几款以前没有用过的软件，如Sphinx, Visual Studio Code, Github。竟然也开始写代码了，感觉很厉害的样子，可以吓唬自己了。这一行怎么这么长啊！原来写代码就是这样啊！

以技会友¶

认识高老师很久了，听过他的讲座，知道他很厉害。但不知道他厉害在那儿？这次上了他的课，总算领教了大神的厉害：由浅入深，循循善诱，十分耐心地一遍一遍地讲解、演示。老师辛苦啦！还有助教小高老师，智慧超群自不必说，还颜值超高呢。她经常来帮助我解决各种问题，总是手到病除——神一样的存在。谢谢美眉啊！参加这次培训的学员大多来自企业。他们皆有一定基础，也给我提供了很多帮助，十分感谢我的队友。
